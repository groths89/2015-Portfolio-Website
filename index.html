Stages of Web Design

1

Planning

To ensure that everyone is on the same page this step is essential to creating a website. This planning phase includes making a paper prototype wireframe, which is used throughout the entire process.

2

Designing

Taking the prototype from the planning stage and making it look good on the computer. It's done by using an advanced image editor to get the drawing to come to fruition.

3

Developing

Using the design I have created and taking it to the next level using web languages to make it function as you see it on the drawings is the last step before getting to the final product.

4

Marketing

Finishing the project by making the website searchable for users to view and buy your products. This uses key words and descriptions so the search engine can evaluate and decide whether to show your site.
